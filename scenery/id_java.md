# Java Island photo scenery

## Description

Photo scenery covering a large part of Java island, particularly the capital city of Jakarta in the west, as well as many other cities such as Badung, Cirebon, Semarang, Surakarta in the middle and Surabaya and Malang in the east and their airports. Discover the island's many volcanoes as you fly from town to town.

It also has a few POI's in Jakarta, Bandung and Surabaya. An elevation fix was made especially for Jakarta airport and town.
