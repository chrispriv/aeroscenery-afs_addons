# Nakhchivan Photo Scenery

<img class="hero-image"
src="../assets/images/scenery/az_nakhchivan/hero.jpg">

## Description

Photo scenery in HD covering Nakhchivan town, enclave of Azerbaijan and the surrounding area.

## Note
To add currently missing pushback parking positions to UBBN Nakhchivan airport, install the add-on created by @Wingberry. It includes approximately 150 airports worldwide (UBBN Nakhchivan has been added).
