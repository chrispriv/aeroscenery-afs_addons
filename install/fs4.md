# FS4 Desktop Installation

## Installation

1. Download the scenery packages.

2. Unzip the following folders:

- `_images`
- `_fs4`

3. Copy unzipped _fs4 folder into your Aerofly FS4 user scenery folder and then palce the _images into your new scenery folder:

```text
Documents\Aerofly FS4\addons\scenery\<scenery-name>\
```

## Notes
Existing files can safely be overwritten.
Depending on scenery size, first loading may take some time.
Large sceneries may require significant disk space.

## Troubleshooting

If scenery tiles do not appear:
- Verify correct folder placement
- Restart Aerofly FS4
- Check available disk space
