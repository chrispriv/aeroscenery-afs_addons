# FSG Mobile Installation

## Installation

1. Download the scenery packages:

- `_images.tme`
- `_fsg.tme`

2. Copy both files to the following directory on your Android device:

   ```text
   /storage/emulated/0/Android/data/com.aerofly.aeroflyfsg1/files/<tme-files>/
   ```
You can use:

- Google Files App
- ZArchiver
- USB cable connection to a PC

## Important Notes
- This is unofficial third-party content.
- Use at your own risk.
- Large sceneries may impact performance on weaker devices.

## Troubleshooting

If you encounter issues or performance problems:

- Remove the .tme files
- Restart the application
