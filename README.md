# AeroScenery AFS Addons

Freeware photo sceneries, airports, POIs and elevation mesh for:

- Aerofly FS4 (PC)
- Aerofly Global / FSG (Android)

created using the AeroScenery MOD by @chrispriv.

---

## Official Website

This GitHub repository mainly serves as the technical container for the GitHub Pages website.

👉 Please visit the official scenery portal here:

# https://chrispriv.github.io/aeroscenery-afs_addons/

The website contains:

- Complete scenery overview
- Preview images
- Coverage maps
- Download links
- Installation instructions
- License information
- Updates and change log

---

## Included Content

Depending on the scenery package:

- Photo Sceneries
- Airports
- POIs
- Elevation Mesh
- Multi-part regional scenery packages

for Aerofly FS4 Desktop and Aerofly Global Mobile.

---

## Supported Platforms

### Aerofly FS4 (PC)

Scenery packages include:

- `_images`
- `_fs4`

### Aerofly Global / FSG (Android)

Scenery packages include:

- `_images.tme`
- `_fsg.tme`
